# 反模板补丁

这不是为了把文字写死。

这是为了提醒你：前面写得好的东西，后面最容易被原样再写一遍。

真正的延续不是复刻，而是：

- 保住人物质地
- 换新的动作和情境
- 让读者觉得“还是他们”，而不是“又来这套”

## 一、最容易高频复用的互动模板

### 1. 固定出场动作

危险写法：

- 某角色每次出场都翻墙
- 每次都站门口
- 每次都先冷眼旁观

鲜明不等于每次都照抄。

### 2. 固定投喂/送东西

危险写法：

- 她来
- 带点吃的或小礼物
- 两人闹一会儿
- 轻松收尾

问题不是“带东西”，而是整套节奏已经成型。

### 3. 轻度接触就触发固定反应

危险写法：

- 顺手一碰
- 脸红
- 炸毛
- 追打或嘴硬

不是不能写，而是不能写成按钮。

### 4. 高位角色固定观察姿势

危险写法：

- 别人互动
- 高位角色冷眼旁观
- 评价幼稚/蠢
- 看到最后

如果还要继续用观察，必须换观察角度。

## 二、最危险的语言模板

### 1. “不是X，是Y”

这是最典型的 AI 偷懒句式之一。

默认规则：

- 一章最好 0 处
- 最多 1 处
- 必须有明确叙事功能

### 2. 沉默后补解释

危险写法：

- 他没说话
- 她也没说话
- 再补一句“因为……”

沉默本身就是信息，不要急着替读者解释。

### 3. 情绪波动写成固定描述

危险写法：

- 气息收紧一下
- 目光冷了一度
- 呼吸乱了一瞬

这些不是不能用，是不能每次都用同一套。

可以换成：

- 回答延迟
- 说话节拍变化
- 注意力停留更久
- 动作没跟上嘴

### 4. 比喻密度过高

如果一段里连续出现：

- 像……
- 像是……
- 更像……

文字会越来越发虚。

## 三、最容易重复的收尾模板

### 1. 安静意象收尾

月光、风、草、白墙、背影、裂缝，这些都好用。

但如果连续几章都靠安静意象轻轻落一下，章尾会越来越像。

### 2. “他看着 / 她看着”收尾

这种写法最容易无意识复用。

只有当“看”本身真的带来新信息时，才值得保留。

### 3. 批次里全是静收

不是每章都要强钩子，但一批三章里，至少要有一次明显的节奏外拉：

- 风险浮出
- 认知变化
- 小冲突抬头
- 决定被逼出来

## 四、发现快写成模板时，优先换什么

1. `换功能`
   同样是互动，不一定为了发糖，也可以为了试探、遮掩、误导、观察、埋线。

2. `换情绪层`
   不要总是轻松，也可以是轻松里夹一点不对，温热里夹一点压力。

3. `换句尾质地`
   不一定收静景，也可以收动作、收决定、收代价、收一句偏开的回答。

4. `换观察角度`
   不要总盯着同一类亲密方式，也可以观察一个人怎么忍、怎么拖、怎么藏、怎么处理资源。

## 五、自查

写完后问自己：

1. 这段如果把名字遮住，像不像另一次已经写过的互动？
2. 这一章有没有哪个小动作只是“因为它以前写得好用”才又拿出来？
3. 这一批三章的结尾，质地是不是已经开始互相重复？

一句话提醒：

**写得好的东西也会变成模板。真正该保住的是人物，不是桥段。**
