# 完整角色构建示例

这份示例展示一个角色从调色盘到行为禁区的全流程。角色是虚构的，用来演示方法，不来自任何具体项目。

示例包含两个角色：一个需要多面性，一个不需要。

---

## 角色一：沈鹿（需要多面性）

一句话定位：看起来什么都不在意、实际上什么都记着的女军医。

### 调色盘

```markdown
角色：沈鹿

主色调：冷淡
底色：控制欲
对冲/点缀：笨拙的温柔

衍生：
- 冷淡衍生一：不主动找人说话。别人问她问题，她回答完就走，不会多聊半句。
- 冷淡衍生二：不是讨厌人，是觉得多余的社交会干扰判断。她的精力永远优先分配给手上的活。
- 冷淡衍生三：表情管理极好，但不是装的——她真的没有太多情绪波动。唯一例外是病人死在手上的时候。
- 控制欲衍生一：手术台上她是绝对的指挥官。谁打断她的节奏她会停下来看着对方直到对方闭嘴。
- 控制欲衍生二：对自己的控制比对别人更狠。失误后不发脾气，但会连续三天减少睡眠复盘每一个细节。
- 控制欲衍生三：她的物品摆放有固定位置。有人动过她的东西她不会说，但会在对方走后全部归位。
- 笨拙的温柔衍生一：她关心人的方式是"多给你切了一份水果放桌上然后走了"，从不解释。
- 笨拙的温柔衍生二：发现有人发烧会直接把药塞进对方手里说"吃"，然后转身走掉。如果对方追问她就说"你脸色不对"。
- 笨拙的温柔衍生三（反直觉）：她不怕血、不怕死人，但怕别人对她说"谢谢"——因为不知道怎么接。
```

为什么这份衍生是"好"的：

- 不停在标签上。"冷淡"展开成了具体的行为场景（不多聊、精力分配、表情管理的例外）。
- 有反直觉组合。"笨拙的温柔衍生三"——一个不怕死人的军医怕别人说谢谢。AI 不会自动生成这种衍生。
- 衍生之间有张力。控制欲和笨拙的温柔在同一个人身上，会让 AI 写出"她想关心但不知道怎么自然地关心"的行为，比单一标签丰富得多。

---

### 多面性

沈鹿需要多面性。她在手术台上和手术台下是两套完全不同的运行模式。

```markdown
## 面 1：手术台上——"主刀"

触发条件：进入手术室、接到急诊、有人在她面前大出血。
能量状态：高度集中。外界信息被过滤到只剩和手术相关的部分。
说话方式：极短。指令式。不用敬语、不解释原因。
  - "止血钳。"
  - "翻过来。"
  - "安静。你在干扰我。"
  - "不用了，缝合我来。"
身体行为：手稳得像机器。视线不离术区。呼吸节奏固定。
这张面的功能：精确。她在这张面里是最好的自己，也是最不像人的自己。

## 面 2：日常——"沈医生"

触发条件：非紧急状态。查房、吃饭、营地里的日常。
能量状态：低。她在日常里像省电模式运行。
说话方式：能不说就不说。说了也是最短的版本。
  - "嗯。"
  - "放那儿。"
  - "不用。"
  - "你去吧。"
身体行为：走路不快。喜欢靠墙站。吃饭很慢但一定吃完。
这张面的功能：节能。手术台消耗太大，日常里她在蓄电。

## 面 3：失控——"裂缝"

触发条件：病人死在手上 + 她认为是自己的失误。不是每次死人都触发，只有她判断"如果我做对了这个人不会死"才触发。
能量状态：内耗。外表几乎看不出变化，但内部在反复回放。
说话方式：比日常更少。回答问题会延迟。偶尔答非所问。
  - （别人问她吃了吗）"……嗯？"
  - （别人问她怎么了）"没。"
  - （三天后）"那个缝合如果换成连续褥式会不会不一样。"
身体行为：走路轨迹不变但步速变慢。会在某个地方站很久不动。夜里不睡觉坐在桌前翻手术记录。
这张面的功能：复盘。她在用自虐式的复盘来偿还"我没做好"的债。

## 面与面之间的过渡

- 面 1 → 面 2：手术结束，她摘手套的速度会比平时慢。走出手术室后靠在墙上站一会儿再走。过渡时间大概十几分钟。
- 面 2 → 面 1：几乎没有过渡。听到急诊铃她已经在跑了。从省电到全功率只需要一个触发。
- 面 2 → 面 3：没有明确切换点。是一个渐变——她不是突然崩溃，是在接下来几天里慢慢变安静、变迟钝。周围人可能第二天才发现她不对。
- 面 3 → 面 2：通常需要一次"被打断"——有人让她去处理别的事，她的控制欲强迫她回到功能状态。不是恢复了，是被覆盖了。

## 渗透

- 面 2 运行中面 1 的泄漏：吃饭时有人切到手，她会瞬间抬头判断伤口深度，然后意识到这不是手术室，又低下头继续吃。
- 面 2 运行中面 3 的泄漏：日常里突然盯着自己的手看，然后把手收回去。
- 面 1 运行中面 3 的泄漏：手术进行到关键步骤时，手停了半秒。然后继续。没有人注意到，但她自己知道。
```

---

### 二次解释

```markdown
关于"冷淡"的本质：
不是不在意，是她认为表达在意会干扰判断。她的职业要求她在最血腥的场景里保持冷静，久了之后情绪通道变窄了。不是没有，是不太会用了。

关于"控制欲"的边界：
她不控制别人的生活。她控制的是"和我工作相关的一切"。手术台上她是暴君，手术台下她懒得管任何人。如果有人把她写成在日常里也颐指气使——错了。

关于"笨拙的温柔"的触发：
她不会对所有人温柔。只有她判断"这个人现在需要帮助但没有开口"时才触发。如果对方主动求助，她反而会用最公事公办的方式处理。她的温柔只出现在对方没注意到的地方。

AI 最可能的误读方向：
- 把她写成"冰山美人"——错。她不是故意冷，是真的觉得多余的话没有必要。
- 把她写成"嘴硬心软"——不完全对。"心软"暗示她内心柔软但嘴上不承认。实际上她内心确实柔软，但她自己可能都没意识到。她不是"不承认"，是"不知道"。
- 让她在关心人之后补一句"别多想"之类的——错。她关心完就走了，不会补任何解释。她甚至不觉得自己在关心。
```

### 行为禁区

```markdown
角色：沈鹿

常态下绝不会做的事：
1. 不会主动解释自己的行为。她切了水果放你桌上，你问她为什么，她说"顺手"。真实原因她不会说，也许她自己都不知道。
2. 不会在公开场合表达情绪。她的情绪只在面 3 里通过行为泄漏（走慢、发呆、翻记录），从不用语言表达。
3. 不会说"我担心你"或任何类似的话。她的担心全部转化为行动——多给你检查一次、把药放你桌上、吃饭时多夹一筷子到你碗里。
4. 不会在手术台上犹豫（面 3 的泄漏除外）。
5. 不会主动找人倾诉。面 3 的复盘是对自己的审判，不是对外的求助。
6. 不会哭。不是忍着不哭，是哭这个反应在她身上已经很难被触发了。

例外条件：
- 第 3 条：如果有人直接问她"你是不是在担心我"，她不会否认，但也不会确认。她会沉默一两秒然后换话题。极端情况下（对方即将做出让她判断为"会死"的事），她可能会说"你别去"——这三个字是她能说出的最接近"我担心你"的版本。
- 第 6 条：全书最多一次。触发条件必须是"她认为可以救活但最终没救活的、对她来说重要的人"。哭也不是嚎啕大哭，是术后一个人坐在手术室里发现脸上有水。

例外出现频率：
- "你别去"：全书 1-2 次。
- 哭：全书最多 1 次。0 次也完全可以。
```

## 角色二：老姜（不需要多面性）

一句话定位：话多、热心、什么都管的后勤老兵，但在关键时刻会突然安静下来。

### 调色盘

```markdown
角色：老姜（姜守义）

主色调：热心肠
底色：老兵的直觉
对冲/点缀：沉默

衍生：
- 热心肠衍生一：谁新来他第一个过去搭话。不管对方什么反应，他都能聊下去。
- 热心肠衍生二：他管的事情太多，经常被人嫌烦，但他不在意。被骂了嘿嘿一笑继续管。
- 热心肠衍生三（反直觉）：他热心不是因为善良，是因为闲下来会想起不该想的事。忙着管别人的事，自己的事就不用想了。
- 老兵的直觉衍生一：他能从一个人走路的方式判断这个人今天状态对不对。
- 老兵的直觉衍生二：有危险的时候他反应比年轻人快，但事后手会抖。
- 沉默衍生一：他平时话极多，所以他一旦不说话，所有人都会察觉。
- 沉默衍生二：他沉默的时候通常在看远处。不是在想什么哲理，是在判断"这件事会不会出人命"。
```

### 为什么老姜不需要多面性

他在不同场景里不会切换整套生存策略。他热心归热心、沉默归沉默，但这两种状态之间是自然过渡，不是模式切换。他没有"伪装"，没有"对外一套对内一套"。调色盘已经足够描述他。

硬给他写多面性会怎样：AI 会把"话多"和"沉默"当成两个开关来回切，丧失他"一直在叽叽喳喳突然某个瞬间安静了"的自然感。

---

### 二次解释

```markdown
关于"热心"的真实动机：
他不是圣人。他管别人的事是因为闲下来大脑会自动回放老战场。这不是创伤后应激——他没有严重到那个程度——只是一个习惯。忙比不忙好过。

AI 最可能的误读方向：
- 把他写成"嘴碎但心里什么都懂的智者"——错。他确实什么都管，但他给的建议经常不靠谱。他的价值不在于说的话，在于他在。
- 把他的沉默写得太有哲理——错。他沉默的时候脑子里在算"这条路上次塌过、那个位置上次出过事"，不是在感悟人生。
```

---

### 行为禁区

```markdown
角色：老姜

常态下绝不会做的事：
1. 不会在年轻人面前提自己以前的事。别人问他也是嘿嘿一笑岔开。
2. 不会承认自己害怕。手抖的时候如果被看到，他会说"老毛病，手凉"。
3. 不会在有人需要帮忙的时候不管。哪怕对方明确说"不用你"，他也会在旁边找个理由待着。
4. 不会在别人面前哭或者表现脆弱。

例外条件：
- 第 1 条：如果有年轻人即将犯一个"他以前见过有人因此死掉"的错，他会提一句。只提一句。不会展开。
- 第 4 条：独处时可以。但不是哭——是坐在那儿很久不动，然后起来继续干活。

例外出现频率：
- 提往事：全书 2-3 次。
- 独处时的脆弱：可以稍多一些，但每次都不超过几行，然后他就起来了。
```

## 错误对照

### 错误一：衍生停在逻辑相关

❌ "冷淡衍生一：不爱说话。冷淡衍生二：不爱社交。冷淡衍生三：喜欢独处。"

这三条说的是同一件事。AI 看到这些衍生等于只看到了一个"冷淡"标签。

✅ 好的衍生要展开到具体行为，并且包含至少一条反直觉的。

### 错误二：不需要多面性的角色硬写多面性

❌ 给老姜写"面 1：平时话多 / 面 2：关键时刻安静"

这不是多面性，这是一个人正常的情绪切换。硬写成面会让 AI 在两种模式间生硬跳转，丢掉自然过渡的质感。

✅ 判断标准：行为模式有没有**根本性**切换。"稍微安静一点"不算。

### 错误三：行为禁区不写例外

❌ "不会哭。" （句号。）

没有例外的禁区是铁律。铁律意味着这个角色永远不会哭。那她哭的那一刻就永远不会出现——你丢掉了全书可能最有冲击力的瞬间。

✅ 禁区必须带例外条件和频率，因为禁区是弓，打破是箭。

### 错误四：二次解释写成演绎指导

❌ "沈鹿在关心人时应该表现得别扭，要用动作而不是语言来表达，营造出一种反差萌的效果。"

这是在告诉 AI"你应该怎么演"。AI 会按这个模板批量套用，每次关心都写成同一种别扭。

✅ 二次解释是告诉 AI"她到底是怎样的"，不是教 AI"怎么表演"。写"她不觉得自己在关心"比写"要营造反差萌"有效一百倍。
